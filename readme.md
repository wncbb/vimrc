## vim language server client
1. 安装coc.nvim
coc.nvim https://github.com/neoclide/coc.nvim
2. 安装llvm
MacOS需要安装llvm，路径在
/usr/local/Cellar/llvm/11.1.0/bin
用到clangd，需要将这个目录放到PATH里
3. 安装对应的语言
language list: https://github.com/neoclide/coc.nvim/wiki/Using-coc-extensions
